# 📅 Школьный Календарь

Система управления школьным расписанием с автоматической системой замен учителей.

![React](https://img.shields.io/badge/React-19.2-blue)
![Vite](https://img.shields.io/badge/Vite-7.2-646CFF)

## ✨ Возможности

- 📅 **Календарь расписания** - месячный вид с отображением всех событий
- 🔄 **Система замен учителей** - автоматический подбор свободных учителей
- 👥 **Панель "Кто свободен?"** - быстрый просмотр доступности учителей
- 🔔 **Редактор расписания звонков** - настройка времени уроков
- 🎯 **Кружки и факультативы** - добавление внеурочных занятий
- 🎨 **Цветовая визуализация** - зеленый/красный/фиолетовый для разных типов событий

## 🚀 Быстрый старт

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`
