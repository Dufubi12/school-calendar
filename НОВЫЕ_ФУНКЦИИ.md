# 🆕 Новые функции календаря замен

## Что добавлено:

### 1. **Панель "Кто свободен?" 👥**
- **Кнопка доступа**: На главной странице календаря
- **Функционал**:
  - Выбор конкретного урока (1-8)
  - Фильтрация по предмету
  - Показывает всех учителей с цветовой индикацией:
    - 🟢 **Зеленый** = Свободен
    - 🔴 **Красный** = Занят
  - Отображает нагрузку каждого учителя (процент и количество уроков)
  - Статистика: сколько учителей свободно/занято
  - Сортировка: сначала свободные, потом по нагрузке

### 2. **Улучшенная модалка замен 🔄**
- **Автоматический подбор свободных учителей**:
  - Система автоматически проверяет занятость по времени
  - Показывает конфликты, если учитель занят
  - Цветовая индикация:
    - 🟢 Зеленый фон = Свободен
    - 🔴 Красный фон = Занят
- **Умная сортировка**: Свободные учителя показываются первыми
- **Предупреждения**:
  - ⚠️ Если выбран занятый учитель, появляется предупреждение
  - Кнопка меняется на "Сохранить (с конфликтом)"
- **Визуализация нагрузки**: Прогресс-бар показывает текущую нагрузку
- **Детальная информация**:
  - В какое время занят
  - Какой урок ведет
  - Номера уроков в этот день

### 3. **Улучшенная визуализация в календаре 📅**
- **Иконки событий**:
  - 🔄 = Замена
  - ✓ = Обычный урок
  - 📖 = Другое событие
- **Цветовая схема**:
  - 🔴 Красный = Замены
  - 🟢 Зеленый = Обычные уроки
  - 🔵 Синий = Другие события
- **Время урока**: Отображается в каждом событии
- **Hover эффекты**: При наведении событие увеличивается
- **Tooltip**: Полная информация при наведении

### 4. **Проверка конфликтов**
- Автоматическая проверка пересечения времени
- Использует точное время начала и окончания урока
- Проверка работает как для обычных уроков, так и для замен
- Визуальное предупреждение о конфликтах

---

## 🎯 Как использовать:

### Быстрый поиск замены:
1. Нажмите **"👥 Кто свободен?"**
2. Выберите **урок** (например, 3 урок: 10:30 - 11:15)
3. (Опционально) Выберите **предмет**
4. Увидите список всех учителей:
   - 🟢 Свободные показаны сверху
   - 🔴 Занятые показаны снизу
5. Кликните на свободного учителя для быстрого выбора

### Создание замены:
1. Кликните на **день в календаре**
2. Выберите **"Добавить замену"**
3. Выберите **отсутствующего учителя**
4. Система автоматически:
   - Покажет подходящих учителей (того же предмета)
   - Проверит их занятость
   - Отсортирует по доступности
5. Выберите **свободного учителя** из списка
6. Сохраните замену

## 📊 Индикаторы:

### Цветовая система занятости:
- **🟢 Зеленый** (0-59% нагрузки) = Нормальная нагрузка
- **🟡 Желтый** (60-79% нагрузки) = Высокая нагрузка
- **🔴 Красный** (80-100% нагрузки) = Перегрузка

### Статусы учителя:
- **✅ Свободен** = Нет уроков в это время
- **❌ Занят** = Уже ведет урок в это время
- **Показывается причина**: "Математика (10:30-11:15)"

---

## 🔍 Технические детали:

### Проверка конфликтов:
Система проверяет пересечение времени по формуле:
```
Конфликт = (НачалоA < КонецB) И (КонецA > НачалоB)
```

Это означает, что даже если уроки начинаются в разное время, но пересекаются - система обнаружит конфликт.

### Данные сохраняются в:
- `localStorage` браузера
- Ключи:
  - `school_calendar_events` - события и замены
  - `school_calendar_teachers` - учителя
  - `school_calendar_timeslots` - временные слоты
